* `^<space>`搜索历史

## Google

| 语法       | 示例                 | 解释                        |
| :--------- | :------------------- | :-------------------------- |
| `site:`    | `site:github.com`    | 在指定域名内搜索            |
| `""`       | `"tag indicator"`    | 精确匹配                    |
| `*`        | `git * --hard HEAD`  | 通配符匹配                  |
| `-`        | `hashcat -wikipedia` | 排除减号后的关键字wikipedia |
| `intitle:` | `dwm intitle:tag`    | 标题包含关键字tag           |
| `inurl:`   | `dwm inurl:tag`      | url包含关键字tag            |
| `intext:`  | `dwm intext:tag`     | 内容包含关键字tag           |
